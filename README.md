# AI IGCSE Question Paper Generator

AI-based web application that generates IGCSE-style question papers based on previous exam papers.

## Features
- AI generated questions
- Based on previous exam papers
- Preview question paper
- Watermark customization
- Export paper as PDF

## Technologies
- Python
- Flask
- Transformers (NLP)
- HTML
- CSS
- JavaScript
- MySQL
- ReportLab

## Installation

Clone the repository

git clone https://github.com/Jasiya/ai-igcse-question-paper-generator.git

Install dependencies

pip install -r requirements.txt

Run application

python app.py
